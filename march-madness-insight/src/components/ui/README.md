# shadcn/ui components

Pruned set used by this app, colocated at `src/components/ui/` (standard shadcn layout).

- **Imports:** `import { Button } from "@ui/button";` — Vite/TS alias `@ui` → `src/components/ui`.
- **CLI:** `components.json` maps `"ui": "@/components/ui"`. Add new primitives with `npx shadcn@latest add <name>` from this directory.
